# Tetris Game (C++ | SFML)

A graphical implementation of the classic **Tetris** game built using **C++** and **SFML (Simple and Fast Multimedia Library)**.
This project demonstrates core programming concepts, real-time game logic, and event-driven design.

---

## Features

* Graphical User Interface (GUI)
* Classic Tetris gameplay
* Main Menu system
* Pause & Resume functionality
* High Score saving using file handling
* Ghost (Shadow) piece for better placement
* Hard Drop & Soft Drop mechanics
* Smooth controls & animations

---

## Technologies Used

* **Language:** C++
* **Library:** SFML
* **Concepts:**

  * Object-Oriented Programming
  * Game Loop
  * Collision Detection
  * File Handling
  * State Machine (Menu, Play, Pause, Game Over)

---

## Controls

| Action       | Key             |
| ------------ | --------------- |
| Move Left    | Left Arrow / A  |
| Move Right   | Right Arrow / D |
| Rotate       | Up Arrow / W    |
| Soft Drop    | Down Arrow / S  |
| Hard Drop    | Spacebar        |
| Pause        | P               |
| Menu Select  | Keys 1 – 4      |
| Exit to Menu | Esc             |

---

## Scoring System

* 1 Line → 10 points
* 2 Lines → 30 points
* 3 Lines → 60 points
* 4 Lines → 100 points
* Level increases after every 5 lines (speed increases)

---

## How to Run

1. Install **SFML**
2. Clone this repository:

   ```bash
   git clone https://github.com/your-username/tetris-sfml.git
   ```
3. Open the project in **Visual Studio**
4. Link SFML libraries properly
5. Build & Run
